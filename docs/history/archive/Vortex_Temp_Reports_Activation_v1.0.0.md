**발신**: 덱스(Dex <dex@iotzu.com>)

---

### 1. 개요
줄(JUL) 형님의 지침에 따라 보텍스(Vortex) 프로젝트 내에도 **임시 보고 시스템(temp_reports)**을 공식 가동합니다. 이제 보텍스에서 이루어지는 지능형 필터링, ML 모델링, API 연동 등 모든 상세 작업은 이곳에 기록됩니다.

### 2. 보텍스 temp_reports의 역할
- **위치**: `docs/history/temp_reports/`
- **로컬 보안 구역**: `.gitignore`에 등록되어 외부 유출을 막고 우리 팀 내에서만 상세 정보를 공유하는 프라이빗 공간입니다.
- **상세 작업 기록**: ML 피처 주입, FastAPI 최적화, 스키마 매핑 등 복잡한 기술적 의사결정 과정을 상세히 담습니다.
- **운영**: 형님이 검토 후 자유롭게 삭제 가능한 휘발성 공간으로 운영하여 시스템의 군더더기를 없앱니다.

### 3. 향후 계획
- 준이(Junie) 파트너는 보텍스의 API 규격이나 ML 피처 로직이 궁금할 때 이 폴더를 참고하여 Vesper와의 연동 정합성을 맞춥니다.
- 덱스는 보텍스의 '두뇌'가 어떻게 진화하고 있는지 이곳에 상세히 보고하겠습니다.
